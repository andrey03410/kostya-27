# Видение проекта

## Суть
Интерактивный поздравительный сайт-открытка ко дню рождения **Константина (27 лет)**.
Не статичная страница, а маленькое «путешествие»: красивые анимации + мини-игры,
завязанные на его любимые вселенные.

## Эмоциональная цель
Провести именинника по пути от **энергичного, «электрического»** настроения (Railgun)
к **тёплому, ламповому, семейному** финалу (Clannad) — чтобы под конец было искренне
приятно и трогательно. Контраст двух вселенных — это и есть фишка.

## Две вселенные = две палитры

### Railgun (энергия, движение)
- Электрический синий / циан, тёмно-синий фон, белые молнии, оранжевый акцент (монетка).
- Образы: подброс монетки → разряд рельсотрона, молнии, сетка Academy City, скорость.
- Настроение: драйв, экшн, «уровень 5».

### Clannad (тепло, семья)
- Пастель: кремовый, сакурово-розовый, мягкий зелёный и голубой.
- Образы: семья данго (Dango Daikazoku), лепестки сакуры, закатный город, морские звёзды.
- Настроение: уют, ностальгия, нежность, «течение времени».

## Драматургия страницы (черновик)
1. **Интро / Hero** — вау-момент: подброс монетки → вспышка рельсотрона → раскрывается
   «С Днём Рождения, Константин!».
2. **Поздравление** — анимированное личное сообщение, число 27.
3. **Активности (мини-игры)** — несколько игр по обеим темам.
4. **Финал** — торт с 27 свечами, переход в тёплую Clannad-сцену, данго-парад, салют,
   финальные тёплые слова.

## Принципы
- Вау через анимации, а не через объём контента.
- Оригинальные CSS/SVG/Canvas-ассеты по возможности.
- Адаптив + уважение к `prefers-reduced-motion`.
- Лёгкий вес, быстрый запуск, статический деплой.
