# CLAUDE.md — инструкции для работы над проектом

## Что это за проект

Поздравительный сайт ко **дню рождения Константина** (исполняется **27 лет**).
Чисто фронтенд, без бэкенда — статический сайт, который можно открыть в браузере
и задеплоить на статический хостинг (GitHub Pages / Netlify / Vercel).

Главная задача — **очень красивые анимации** и **интерактивные активности
(мини-игры)**, объединённые двумя любимыми вселенными именинника.

### Тематика (важно!)
Константин любит:
- **Clannad** (визуальная новелла + аниме) — тёплая, ламповая, семейная эстетика.
  Ключевые образы: данго (だんご大家族 / Dango Daikazoku — семья данго), сакура,
  пастельные тона, тема семьи, города и течения времени, Нагиса, Фуко (морские звёзды).
- **A Certain Scientific Railgun / とある科学の超電磁砲** — энергичная, «электрическая»
  эстетика. Ключевые образы: Мисака Микото, подбрасывание монетки → выстрел рельсотрона,
  молнии, синие электрические разряды, Academy City, эспер уровня 5.

Дизайн-концепт строится на **контрасте и переходе** между тёплой пастельной частью
(Clannad) и холодной электрической частью (Railgun).

## Технологический стек (финализирован)

- **Vite + React** (JSX, уже инициализировано)
- **Framer Motion** — основные анимации и переходы
- **Tailwind CSS** — стилизация (+ токены из `theme.css`)
- **Canvas / tsParticles** — частицы (молнии, лепестки сакуры, данго)
- **Howler.js** — управление звуком/музыкой
- **Zustand** или React Context — лёгкое состояние (счёт в играх, прогресс)

> См. журнал решений в `.ai/decisions.md`.

## Формат сайта (решено)
- **Скролл-сторителлинг**: одна страница, секции сверху вниз, анимации по скроллу.
  Поток: Интро ⚡ → Поздравление → Хаб игр 🎮 → Финал 🌸🎂.
- **Мини-игры (4)**: Рельсотрон-тир, Лови данго, Мемори/найди пару, Аниме-квиз.
- **Музыка**: тумблер, по умолчанию выключена (без автоплея).
- **Текст поздравления**: пока плейсхолдер от Claude, заменим позже.

## Структура проекта

```
src/
  assets/       — изображения, звуки, шрифты
  components/   — переиспользуемые UI-компоненты
  sections/     — крупные секции страницы (Hero, Greeting, Games, Finale)
  games/        — мини-игры (по папке на игру)
  hooks/        — кастомные хуки
  data/         — данные (вопросы квиза, тексты поздравлений)
  styles/       — глобальные стили и токены темы (theme.css)
.ai/            — проектная документация и декомпозиция по фазам
```

## Принципы работы

1. **Оригинальные ассеты по возможности.** Данго, молнии, монетку, сакуру рисуем
   через CSS/SVG/Canvas — это и красиво, и без проблем с авторскими правами.
   Готовые картинки/музыку из аниме используем осторожно (личный некоммерческий подарок).
2. **Производительность анимаций** — отдаём предпочтение `transform`/`opacity`,
   уважаем `prefers-reduced-motion`.
3. **Адаптивность** — сайт смотрим и с десктопа, и с телефона.
4. **Текст — на русском** (если не решим иначе).
5. **Идём по фазам** из `.ai/phases/`. После завершения фазы — отмечаем статус в файле фазы.
