# .ai — проектная документация

Здесь живут рабочие документы по проекту поздравительного сайта для Константина.

## Файлы

| Файл | Назначение |
|---|---|
| `project-overview.md` | Видение, концепция, тематика, целевая «вау-эмоция» |
| `ideas.md` | Банк идей: анимации, секции, мини-игры (черновик для обсуждения) |
| `tech-stack.md` | Технологии и обоснование выбора |
| `decisions.md` | Журнал принятых решений (ADR-лайт) |
| `phases/` | Декомпозиция работ по фазам |

## Как пользоваться

1. Обсуждаем концепцию → фиксируем решения в `decisions.md`.
2. Идеи копятся в `ideas.md`, лучшие переходят в фазы.
3. Работаем по `phases/` сверху вниз, отмечаем статус (`☐ / ⏳ / ✅`).
4. `CLAUDE.md` в корне — краткая выжимка для ассистента.

> Статус: **черновики**. Содержимое уточняется на этапе обсуждения.
