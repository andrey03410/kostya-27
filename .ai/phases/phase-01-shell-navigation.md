# Фаза 1 — Каркас приложения и навигация

**Статус:** ✅ готово
**Цель:** оболочка сайта, переходы между секциями, фоновая атмосфера, управление звуком.

## Задачи
- [ ] App-оболочка и контейнер секций (скролл или роутер — по решению)
- [ ] Плавные переходы между секциями (Framer Motion)
- [ ] Система фоновых частиц с переключением темы по секциям (Railgun → Clannad)
- [ ] Тумблер музыки/звука (вкл/выкл, громкость)
- [ ] Индикатор прогресса/навигация (точки, меню или стрелка вниз)
- [ ] Поддержка `prefers-reduced-motion`

## Критерий готовности
Можно пройти по всем (пока пустым) секциям с красивыми переходами и атмосферным фоном.
